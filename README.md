# Docker programms

Этот проект представляет собой методическое пособие по усвоению команд в Doccker, чтобы не вводить команды по тысячу раз

## Install

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/Fil-web/docker_exams.git
    cd docker
    ```
2.  Создайте и активируйте виртуальное окружение:
    ```bash
    python -m venv venv
    source venv/bin/activate
    ```
3.  Установите необходимые библиотеки:
    ```bash
    pip install -r requirements.txt
    ```


1. Чтобы начать использовать программу, в открытой папке проекта запустите

./start.sh
