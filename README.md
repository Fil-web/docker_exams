# Docker programms

Этот проект представляет собой методическое пособие по усвоению команд в Doccker, чтобы не вводить команды по тысячу раз

## Install

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/Fil-web/docker_exams.git
    cd docker
    ```

2. Чтобы начать использовать программу, в открытой папке проекта запустите

./start.sh
